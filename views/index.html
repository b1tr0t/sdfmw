Sane Defaults for Optimizing
Mobile Web Performance

John Boxall, Jan 26, 2013

  • Place <script>s near </body> to unblock rendering

    When a browser encounters JavaScript it must stop fetch, parse and execute the script to continue. Place JavaScript at the bottom of your document to avoid blocking the browser and speed up rendering. Demo →

  • Use a CDN to decrease request times

    Content Distribution Networks use multiple endpoints reduce the distance between the browser and the content. This improved pathway reduces the time it takes for the browser to fetch resources. Demo →

  • Add caching headers to static resources to reduce requests

    The fastest request is the one never made. By serving static assets with caching headers you can reduce the number of required to load your page. Demo →
